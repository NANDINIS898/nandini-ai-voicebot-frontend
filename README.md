# Nandini AI Voicebot

This is a **voice-based AI assistant** project built to demonstrate voice interaction capabilities. The bot can respond to user queries with **spoken answers**, providing a natural conversation experience. This project is meant for **demo/interview purposes**.
## To hear my voice :
Type: What is your life story?
what are your super powers?
what is the misconception people have about you?
what are your top 3 growth areas?

and you,ll get to know me a bit !
## Features

* Type a question and **hear the answer** in your own voice.
* AI-powered response generation.
* Plays audio responses seamlessly.
* Lightweight frontend built with ReactJS and utility scripts for voice handling


Run the project locally:

# Navigate to the project folder
cd voicebot

# Install dependencies
pip install -r requirements.txt

# Start the frontend (React)
cd frontend
npm install
npm start

Open the app in your browser.

Type a question in the input field and press Enter. The bot will speak out the answer using your voice.
